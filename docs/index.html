Spatial Tarot
3D web wireframe (Volumetric Cross)

Wireframe goals

  • Test whether a volumetric spread stays interpretable in 3D.
  • Validate the flow: question → draw → place → inspect → annotate → export.
  • Prototype input redundancy: drag/drop + click-to-place (touch-friendly).

Method: Volumetric Cross

Three discrete planes: Understory (hidden), Surface (present), Horizon (trajectory). Slots align vertically so you can read “correspondences” across layers.

What to test (5–10 min)

  1. Enter a question.
  2. Draw 6 cards, place them into any slots.
  3. Inspect a card; add a note.
  4. Hover a slot to see vertical correspondences.
  5. Export JSON and share it with you.

Session

Place cards by dragging onto a slot or by clicking a card, then clicking a slot.

Hand

Tip: “Inspect” brings a card to a stable reading distance (wireframe version).
