# Spatial Tarot 3D

A separate browser-based 3D demo for the Spatial Tarot concept.

## Run

```bash
cd spatial-tarot-3d
npm install --cache /tmp/spatial-tarot-npm-cache
npm run dev
```

## Build

```bash
cd spatial-tarot-3d
npm run build
```

The production build writes to the repository root `docs/` directory so the
site can be published directly with GitHub Pages from `main` at `/docs`.

## What it includes

- Three luminous planes: Understory, Surface, Horizon
- Animated auto-deal into a volumetric spread
- Click-to-inspect correspondence stacks
- Per-card notes
- JSON export
